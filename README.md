A sleek, dark-mode animated portfolio website built as a single standalone index.html file. No frameworks, no build tools, no dependencies — just pure HTML, CSS, and JavaScript.
Aesthetic: Developer Cyberpunk meets Professional Minimalist — deep slate backgrounds, neon blue and purple accents, particle canvas animations, and smooth scroll-reveal transitions.
Sections:

Hero introduction with floating 3D-style avatar and kinetic typography
Verity project showcase — full-stack hackathon app built in 48 hours with a distributed global team
Image Forgery Detection — ML academic project with animated pixel-grid forensics visualizer
Tech stack breakdown across Frontend, Backend, and Data/ML
Call-to-action with contact details and hire links

Tech used in the site itself: Vanilla JS · CSS animations · Canvas API · IntersectionObserver · Google Fonts (Space Mono + Inter)
